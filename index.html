O que deseja fazer ?

  • O que é o SBI ?

    SBI é um sistema voltado para controle de atividades relacionadas à setor de Bilhetagem eletronica muito comum em sistemas de transporte publico, é um sistema que procura resolver os principais problemas relacionados à area, como controle de catraca, gerenciamento dos principais problemas tecnicos encontrados

  • Como usar o sistema ?

    O sistema é simples e intuitivo, mas para o conforto e usabilidade, disponibilizamos um manual em pdf onde todas as duvidas com relação ao uso poderão ser sanadas, caso necessario o usuário poderá entrar em contato conosco atraves dos contatos disponibilizados no proprio sistema e no manual, a unica observação a ser feita é que a unica forma de utilizar o sistema e tendo um usuario e senha para loguin

  • Quero adquirir o SBI!

    Você poderá adquirir o SBI entrando em contato com nossa equipe, os nossos meios de contato estão disponiveis aqui no site e tambem no manual do sistema ao adquirir esse sistema você, terá direito ao uso do mesmo de forma Trial por até tres meses
